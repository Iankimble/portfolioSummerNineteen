Black-U-Weather
Weather applications that returns current forcast and expressive gif.

Summary
During my time attending the University of Pennsylvania's Coding bootcamp, One of our assignments was to create an application that uses two API's (application programming interface) simulataniously with one another. I wanted to design something fun and entertaining but also practical.
Black-u-weather is a weather application that searches American cities and returns the current weather for that location along with a gif that illustrates the "feel" of the weather description.
Tools and Technologies
The basis of this application is wrting logic in javascript to send requests and recieve responses from the open weather API and the Giphy API.
Concept
Challengs and solution
The difficulty in building this application was finding gifs that reflected a specific weather condition. Fortunately, the Open Weather API has a finite list of weather descriptions. With the short list of descriptions, I resolved the issue by creating two functions; One would request the weather information from Open Weahter and return the weather data, specifically the weather description. The other would take the description and compare it to a series of conditional statements that would return a set of specific gifs, so if the weather description returend "cloudy", 1 of 3 gifs that I felt conveyed the feeling of a cloudy day would be returned. and displayed along with the temperature
Regrettably, the app is limited by the usage. Open weather requires a paid subscription to use their weather data publically.
